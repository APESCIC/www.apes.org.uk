# APES CIC optional Remotion motion pack

This folder contains an optional Remotion source pack for APES hero-loop and campaign motion exports.

## Purpose

- Support future homepage or landing-page motion without making Remotion a runtime dependency of the public site.
- Reuse the same APES colour system and tone established by the rebuild and Creative Production mood board.

## Suggested usage

1. Install dependencies in this folder when motion production is needed.
2. Run `npm run studio` for iteration.
3. Render `APESHeroLoop` as a hero background loop or campaign teaser.

## Current status

- Source-only
- Not required for Cloudron deployment
- Safe to exclude from the public bundle
