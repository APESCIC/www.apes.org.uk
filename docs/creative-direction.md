# APES CIC creative direction pack

This folder records the Creative Production visual direction used to shape the rebuilt APES site.

## Purpose

- Lock the mood, palette and material language for the APES rebuild without publishing synthetic imagery as if it were documentary APES photography.
- Give future designers, developers and reviewers a consistent visual reference set for hero panels, cards, donation moments, education scenes and governance surfaces.

## Core constraints used

- Palette: Primary teal, Deep teal, Soft mint, Warm sand, Donation coral accents
- Tone: Compassionate, Trustworthy, Practical, Plain-language, Welfare-first
- Visual language: Editorial hero panels, Rounded trust cards, Accessible data points, Soft gradients, Structured callouts
- Avoid: Purple bias, Overcrowded hero banners, Tiny body text, Decorative-only footer, Sales-first messaging
